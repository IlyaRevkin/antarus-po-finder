# Antarus ПО Finder

Десктопное приложение для управления базой прошивок оборудования.  
Разработано для предприятия **Antarus** (antarus.su).

---

## Возможности

- Поиск прошивок по названию шкафа (ПЧ, КПЧ, SMH5, KINCO и др.)
- Загрузка новых версий с авто-определением метаданных
- Управление правилами (карты IO, инструкции, паспорта, параметры ПЧ/КПЧ/УПП)
- Шаблоны параметров ПЧ/КПЧ/УПП с поддержкой расписаний и копирования
- Синхронизация с сетевым диском
- Быстрые приложения (SMLogix и др.) одной кнопкой
- Сканирование протоколов (WIA / Windows Scan)
- Загрузка фото через QR-код со смартфона
- Тёмная и светлая тема
- Роли доступа: Наладчик, Нал-Администратор, Программист, Администратор

---

## Системные требования

- Windows 10 / 11 (64-bit)
- Python 3.10+ (если запуск из исходников)
- Или готовый EXE (без Python)

---

## Быстрый запуск (EXE)

Просто запустите `Antarus_PO_Finder.exe` — никакой установки не требуется.

> **База данных** создаётся автоматически при первом запуске рядом с EXE  
> (`firmware_finder.db`). Не удаляйте этот файл — в нём хранится вся база прошивок.

---

## Установка и запуск из исходников

### 1. Клонировать / скопировать проект

```
C:\MyApps\FirmwareFinderV2\
```

### 2. Установить зависимости

```bash
pip install -r requirements.txt
```

Зависимости:
| Пакет | Назначение |
|-------|-----------|
| `PySide6>=6.6.0` | GUI-фреймворк (Qt6) |
| `Pillow` | Работа с изображениями, генерация иконки |
| `py7zr` | Поддержка 7z-архивов |
| `qrcode[pil]` | QR-коды для загрузки фото |
| `pyinstaller` | Сборка EXE (только для разработчиков) |

### 3. Запустить

```bash
python main.py
```

---

## Сборка EXE

```bash
# Генерация иконки (нужна один раз)
python make_assets.py

# Сборка
build.bat
```

Результат: `Antarus_PO_Finder.exe` в корне проекта и в папке `dist\`.

## База данных

Файл `firmware_finder.db` (SQLite) создаётся автоматически.  
Хранит: прошивки, правила соответствия, шаблоны параметров, настройки.

**Расположение по умолчанию:** рядом с `main.py` / EXE.

---

## Роли и пароли

| Роль | Доступ | Пароль |
|------|--------|--------|
| Наладчик | Только поиск | — |
| Нал-Администратор | Поиск + шаблоны | — |
| Программист | Поиск + загрузка + шаблоны | Настраивается в Настройках |
| Администратор | Полный доступ | Настраивается в Настройках |

Пароли задаются в разделе **Настройки → Общие → Пароли** администратором.

---

## Сетевой диск

Путь к папке с прошивками на сетевом диске задаётся в **Настройки → Общие → Путь к диску**.  
Синхронизация запускается автоматически каждые 5 минут.
